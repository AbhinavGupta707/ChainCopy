# ChainCopy Name Research

Date: 2026-06-19

## Current Recommendation

Keep `ChainCopy` as the working product name for now. It is clear, short, and maps directly to the core behavior. Treat it as a working name until a real trademark search and domain/social handle pass are completed.

## Findings

- `ChainCopy` does not appear to be an obvious existing macOS clipboard product from public web search.
- GitHub search for `ChainCopy` found one public repo named `Abhisheksinha1506/chaincopy`, created 2026-02-20, with no visible product positioning in the CLI result.
- GitHub search for `"chain copy"` surfaced mostly blockchain/copy-trading repositories and one older unrelated file-copying repository, `jkdevg/QtChainCopy`.
- Apple public search for `ChainCopy` returned fuzzy results such as `Tracelocker`, `MetaSuites`, and `Gemino`; no exact `ChainCopy` or `Chain Copy` Mac clipboard app appeared in the returned result names.
- Apple public search for `Chain Copy` returned unrelated copy/link/file-transfer apps, not an exact ChainCopy clipboard utility.

This is engineering-level availability research, not legal clearance.
